
This directory contains model outputs and analysis results:

- `feature_importance.csv` — importance scores from Random Forest  
- `cluster_summary.csv` — average indicators per cluster  
- `future_predictions.csv` — predicted life expectancy for 2025 & 2030  
- `clustered_data.csv` — dataset with assigned cluster labels  
- `life_expectancy_model.pkl` — trained model file
